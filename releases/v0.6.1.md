## Changes from v0.6.0

### Improvements to types

- Move DOM `lib` inclusion to the type definition file, so consumers no longer need the DOM lib in their own `tsconfig.json` (by @Jym77 in #709)

### Website & docs

- Add a table of core maintainers to the homepage and README (by @LeaVerou and @dmitrysharabin in #705)
- Add an escape hatch to correctly render emojis inside clipped text, i.e. inside blocks of text with the `-webkit-background-clip: text` CSS rule applied (by @dmitrysharabin in #702)
